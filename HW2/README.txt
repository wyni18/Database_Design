
To compile the program
    javac -d .\build -classpath .\lib\ojdbc6.jar;.\lib\sdoapi.jar;.\lib\sdoutl.jar;.\lib\simplefan.jar .\src\HW2.java
To run the program
    java -classpath .\lib\ojdbc6.jar;.\lib\sdoapi.jar.\lib\sdoutl.jar;.\lib\simplefan.jar .\build HW2

The class name for the Java program is HW2.java, there are 8 different
arguments for different problems:
For Problem 1, the arguments are like 
    window firebuilding 10 20 300 500

For problem 2, the arguments are like
    within firehydrant OHE 250
	
For problem 3, the arguments are like
    nn building b3 5
	
For demo problems, the arguments are like
    demo 1
	demo 2
	demo 3
	demo 4 
	demo 5
